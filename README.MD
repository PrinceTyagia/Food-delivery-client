# To check the node version

```
node --version
```

# To initialize the Firebase console

```
firebase init
```

# To start the server

```
npm run serve
```

# To create react project using npm & yarn

```
npm create-react-app folder_name
```

```
yarn create-react-app folder_name
```

# To start client

```
npm start
yarn start
```
